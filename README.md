## Environments
```env
R7PLATFORM_WORKER_METADATA_REDIS_HOST=localhost
R7PLATFORM_WORKER_METADATA_REDIS_PASSWORD=admin

R7PLATFORM_WORKER_METADATA_CONCURRENCY=4
R7PLATFORM_WORKER_METADATA_QUEUE=METADATA
```

## Run

```shell
NODE_ENV=development \
R7PLATFORM_WORKER_METADATA_QUEUE=METADATA \
R7PLATFORM_WORKER_METADATA_REDIS_HOST=localhost \
R7PLATFORM_WORKER_METADATA_REDIS_PASSWORD=admin \
npx nodemon
```
